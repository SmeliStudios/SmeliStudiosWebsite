# SmeliStudios Landing Page

A framework-free static site for `smelistudios.com`.

## Preview

Run any static file server from this directory, for example:

```powershell
npx serve .
```

## Deploy

Upload the directory contents to GitHub Pages or any static hosting provider.
No build step is required.

## Branding

The main colors, spacing, radius, and width controls are CSS custom properties
at the top of `styles.css`.
